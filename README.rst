sbx plugin for `Tutor <https://docs.tutor.edly.io>`__
#####################################################

Plugin to customize WGU Open edX sandbox tutor installation.


Installation
************

.. code-block:: bash

    pip install git+https://github.com/WGU-Open-edX/tutor-contrib-sbx

Usage
*****

.. code-block:: bash

    tutor plugins enable sbx
